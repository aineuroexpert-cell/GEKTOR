Изоляция Отчаяния (Tilt-Breaker): При детекции серии убытков система классифицирует Оператора как «Компрометированный Биологический Объект». 3. АРХИТЕКТУРА И ЭШЕЛОНЫ (5 ФАЗ)

ФАЗА 1: ИНГЕСТИЯ И АБСОЛЮТНАЯ ПАМЯТЬ (Zero-Copy Fabric)
True SHM Bridge: Данные стаканов по ВСЕМ монетам Bybit мапятся напрямую в системную оперативную память через ctypes.Structure. Передача слепка ликвидности в квант-движок занимает 0 наносекунд.
Sequence Guard & Heartbeat: Жесткая валидация u-ID (Update ID) и аппликационный JSON-пинг. При обнаружении разрыва пакетов система локализует ущерб и переходит в статус DEGRADED (сброс и ре-гидрация стейта), предотвращая кому Event Loop'а.

ФАЗА 2: КАУЗАЛЬНОЕ СЖАТИЕ И МИКРОСТРУКТУРА (Causal Temporal Join)
$O(\log N)$ Bisect Conflation: Синхронизация потоков ленты сделок (L1) и стаканов (L2) происходит строго по Exchange Timestamp через бинарный поиск в Numpy-массивах. Никаких $O(N)$ циклов. Система безошибочно реконструирует состояние стакана, которое видел маркет-мейкер в миллисекунду удара.
Relative Capitulation Scanner (LFI): Векторный анализ ликвидаций относительно глубины стакана (Impact Ratio) по всей бирже. Радар ждет момента, когда HFT-фонды начнут поглощать "глупые деньги" на аномальных выбросах токсичного объема.

ФАЗА 3: ФИЗИКА ИСПОЛНЕНИЯ (Execution Realism & Normalizer)
$O(N)$ Walk the Book (MSQ Sizer): Алгоритм рассчитывает максимальный безопасный объем (Maximum Safe Quantity) не "на глаз", а физически симулируя проедание L2-стакана слой за слоем, пока потенциальная прибыль не сравняется с комиссионным трением (Total Friction).
Exchange Filter Rules: Генерация абсолютно строгих инструкций для Оператора. Цены округляются в худшую сторону (пессимизация), объемы — строго вниз до минимального лота. Рекомендации Market-ордеров заблокированы. Доступны только: IoC (Immediate or Cancel) или Maker Limit.

ФАЗА 4: ВЫЖИВАНИЕ СТЕЙТА (Deterministic State Recon)
The Oracle Principle (Private WS): Приватный топик order — это абсолютный оракул. Система слушает конкретные orderId. Разрешение суперпозиций (Partial Fills / FIFO Mirage / Shrapnel) происходит детерминированно, математически точно и без паники.
Bulletproof Flight Recorder: Предсмертные дампы памяти и логи пишутся синхронным демоном-наблюдателем, который гарантированно переживет краш основного Event Loop'а. Внедрен таймаут $O(1)$ Time-to-Die для защиты от зависаний при отказе файловой системы.

ФАЗА 5: ТОПОЛОГИЯ СИГНАЛОВ И ИНВАЛИДАЦИЯ (Residual Alpha & Abort)
Leader-Follower Conflation (Гравитационный Якорь): Биткоин (BTC) и Эфир (ETH) — это гравитационный контекст. При резком макро-импульсе на лидере, радар активирует надсистемный предохранитель, фильтрует структурный шум и выдает сигналы только по тем альткоинам, где альфа еще физически не арбитражирована.
Event-Driven Revoke: При развороте вектора дисбаланса (Regime Inversion), сигнал аннулируется немедленно. Если истек Intent TTL (5 сек), локальный стейт разблокируется атомарно, предотвращая REST-штормы исходящих API запросов.
