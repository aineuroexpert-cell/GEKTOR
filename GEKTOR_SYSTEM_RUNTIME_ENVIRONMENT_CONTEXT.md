# GEKTOR SYSTEM RUNTIME ENVIRONMENT CONTEXT (v7.0)
## Главная цель: Охота на среднесрочные аномалии и LFI (Ликвидации) на 500+ контрактах Bybit одновременно.

## Архитектурные ограничения (Нарушение = Саботаж):
1. ИНГЕСТИЯ: Данные L2-стаканов пишутся напрямую в Ring Buffers (Numpy/ctypes). Использование list.append в горячем пути ЗАПРЕЩЕНО.
2. СТЕЙТ: Мутирует только асинхронно по ACK из приватного топика WebSocket. REST-опросы балансов запрещены.
3. ТИПИЗАЦИЯ: Строгий Python 3.11+. Обязательные Type Hints. Все расчеты цен/объемов — строго через decimal.Decimal или Scaled Integers (Float-галлюцинации запрещены).
4. ВЫХОД ИЗ СТРОЯ: При падении Event Loop — немедленный дамп памяти через flight_recorder.py.
